# Test Subskill

## Lancer le projet

Cloner le repository

Créer à la racine un fichier **.env** et mettre ceci en changeant les informations par les vôtres:<br>

    MAIL="example@example.com"
    PASSWORD="password1234"

Se mettre dans un terminal, puis :
* `npm install`
* `composer install`
* `gulp watch`
